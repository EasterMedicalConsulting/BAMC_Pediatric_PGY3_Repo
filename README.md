ABOUT

BAMC Pediatric Residency AI Agent Repository

This repository was developed for PGY-3 Pediatric Residents at Brooke Army Medical Center (BAMC) as an introduction to practical AI-assisted workflows and basic agent development.

The transition from residency to military physician leadership often requires skills that are not formally taught during training. New physicians are expected to function not only as clinicians, but also as officers, administrators, educators, and organizational leaders. Modern AI tools can help accelerate learning, improve efficiency, and serve as force multipliers when used appropriately.

The goal of this repository is to provide simple, real-world examples of how large language models (LLMs) can be configured to act as trusted assistants. Examples included in this repository demonstrate how to create agents that can function as mentors, advisors, administrators, project assistants, and personal knowledge managers.

This repository is platform agnostic. While many examples were developed using Ask Sage, the concepts can be applied to other LLM platforms including ChatGPT, Claude, Gemini, and future AI systems.

The materials within this repository introduce several foundational concepts:

* Creating agent personas
* Building personal context files
* Developing persistent memory structures
* Capturing user preferences and working styles
* Creating reusable prompt frameworks
* Leveraging AI to support professional development and leadership growth

The intent is not to create autonomous systems that replace human judgment. Rather, these examples demonstrate how AI can be used to augment decision-making, improve productivity, and provide structured guidance for common administrative and leadership tasks.

Recommended Approach

Start simple.

Create an agent that understands who you are, what your goals are, and how you prefer to work. Over time, add context, refine prompts, and build memory structures that allow the agent to become more useful.

The most effective agents are rarely the most complex. They are the ones that consistently understand the user, retain relevant context, and help solve real problems.

Focus first on building an assistant that helps you think, learn, organize, and lead more effectively. Once you understand the fundamentals, you can expand into more advanced agent workflows and automation strategies.

Agent Memory Instructions

When reviewing this file:

1. Prioritize recent entries unless older entries remain relevant.
2. Identify recurring themes and patterns.
3. Use lessons learned to improve future recommendations.
4. Highlight conflicts between past and current goals.
5. Recommend updates when information appears outdated.
6. Never modify historical entries without documenting the reason.
7. Prompt the user to update the file asking specifically if any of the following apply

* major career transitions
* significant leadership experiences
* Following major successes or failures
* Quarterly
* First of each year (January 1st)
